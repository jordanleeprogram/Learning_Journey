Fundamentals

Knowing the fundamentals of computing gives you the foundations to build on that knowledge, knowing why and how things are happening.

Components

Inside a computer are transistors to represent the 1s and 0s of binary code by being either on or off.

The motherboard connects all components of a computer together such as the CPU, RAM, hard drive, connectors for input/output etc.

Some of these components are as follows:

There is also RAM (Random Access Memory) which is short-term memory that disappears once the system turns off. RAM is needed on every device.

HDD (Hard Disk Drive), SSD (Solid State Drive) and Optical Drives (CDs/DVDs) have long-term memory and take longer to process information.

HDD is the main data storage, it has the operating system on it.

SSD is faster and more advanced, with no moving parts (unlike HDD) Unlike the volatile nature of RAM, these retain access to the data.

The Power Supply is a hardware component that supplies electricity to the machine, it would be pretty useless without one.

The GPU is a graphics processing unit, it allows graphics to be rendered in an efficient and effective manner.

Servers allow for a centralised storage. This can be fore files, printer jobs, CCTV footage, e-mails, databases etc.
Examples: Web Server- web storage File Server- for computers on same network Print Server- connects to a printer Application Server- like google docs operates and stores online Media Server- files are streamed as opposed to being sent in full

The Router connects to devices to provide internet communication.

Data

Data can be represented in multiple ways, such as words, numbers, images, sounds and bytes. Numeric: (integers and floats) String: ("Characters") Boolean: True / False Lists: [arrays used to store many elements] Data can be stored in several ways. Direct Area Storage(DAS): is the storage local to your specific machine. Network Based Storage: is off-site and can be accessed by different machines.

The Internet

The internet is an example of Wide Area Network (WAN(. It is of the biggest WAN in the world. A Wide Area Network is not limited to a single location, but it spans over a large geographical area through a telephone line, fibre optic cable or satellite links.
A Wide Area Network is widely used in the field of business, government, and education.
A Local Area Network, however, is the common, limited area coverage such the wifi in a home.

The way the data arrives at your computer is through it's IP address, which allows the data to be sent to your unique location.

Transmission control protocol (TCP) drives reliable data transfers while user datagram protocol (UDP) prioritizes speed and efficiency.
But both are a method in which pockets of data are transferred between devices, whilst checking for errors along the way.
Using a Transport Layer Security (TLS) you can send encrypted data - preventing it from being seen by anyone who shouldn't.

In order for your browser to the load the information of webpages you are looking at on the internet, hhtp and dns protocols are used. DNS (Domain Name System) is the system and protocol that translates domain names to IP addresses and more. HTTP (Hypertext Transfer Protocol) is used to transfer webpages.

Network Topologies

A Network Topology is the way in which a collection of systems are connected. Here are a few examples:

Bus Topology

All devices in the network are connected by one central RJ-45 network cable or coaxial cable. The single cable which transmits the data between all devices is referred to as a bus or backbone.

Star Topology

The Star Topology uses a centralised hub to connect to each host simulataneously.

Mesh Topology

The Mesh Topology is a LAN with as many nodes between each client for an efficient data route.

Ring Topology

The Ring Topology has each node connecting to two other nodes from either side, forming a single continuous pathway for data to transmit.

Boolean Algebra

Boolean Algebra uses True or False to represent 1s and 0s respectively.
There are different types of Logic Gates that use boolean algebra, here are some visual representations:


NOT gates take one input and produces one output, this is called an inverter.
Input true = false
Input False = true


AND gates take two or more inputs and produce one output. Returns true if all inputs are true.
Both false = false
One false/ one true = false
Both true = true


NAND gates take two inputs and provides one output, which is inverted.
Both false = true
One false/ one true = true
Both true = false


OR gates takes two or more inputs and produces one output, returns true is one is true.
Both false = false
One false/ one true = true
Both true = true


XOR gate, or exclusive-or, takes two inputs and produces one output. It excludes both outputs being the same.
Both false = false
One false/ one true = true
Both true = false


NOR gates take two or more inputs and produces one output, an inverted-OR.
Both false = true
One false/ one true = false
Both true = false


EXNOR gates, excluvise-nor, can only take two inputs and produce one output. It is an inverted EXOR.
Both false = true
One false/ one true = false
Both true = false
