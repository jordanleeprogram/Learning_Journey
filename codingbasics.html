Coding Basics

There are different types of programming languages, as well different programming languages themselves, but they share many fundamental similaries.
This page has information about different programming languages, different types of programming languages, functions and coding principles.

Programming Languages

HTML:

Hypertext Markup Language is the skeleton of a website.
A markup language puts the structure in place for other languages to be written and displayed.


CSS:

Cascading Style Sheets is used in conjunction to HTML as it styles the elements stated by HTML.
This is like the hair and clothing if HTML is the skeleton.


Javascript:

JS is multi-paradigm, it supports event-drivem, functional and imperative programming styles.
It is used to create interactivity on websites, add functionality, creatre user accounts, send and request the data to the servers and databases and more.


Python:

Python is a multi-paradigm, object-oriented, procedural (imperative), functional programming language.
It can be used for web applications and data-analysis.


SQL:

SQL stands for Structured Query Language and is domain-specific.
It is used in programing to store, manipulate and retrieve data from databses.

Transition example CSS

Hovering over the parent Div (blue field) will make the child Div (red field) animate across the X-axis in 5 seconds.
You will notice that upon removing the cursor from the parent field returns the child Div (red field) back to its original position as the same speed.
This is because I have ALSO added the 'transition:5s' property to the '.child' in CSS, as well.
The colour of the child Div will also gradually change color, as the ':hover' state in CSS has been allocated the color pink (it's almost white) AS WELL AS the transition property. Meaning that it not only traverses the X-axis over 5 seconds, but also changes color over 5 seconds.

Programming Types

Different languages operate in different manners. These different types are classified as programming paradigms. Here are some examples:


Procedural language:

In procedural languages, the program code is written as a sequence of instructions. User has to specify “what to do” and also “how to do” (step by step procedure). These instructions are executed in the sequential order. These instructions are written to solve specific problems.


Object-oriented language:

Object-oriented programming (OOP) is a style of programming characterized by the identification of classes of objects closely linked with the methods (functions) with which they are associated. It also includes ideas of inheritance of attributes and methods.


Imperative programming:

In imperative programming, every operation is coded and the code itself specifies how the problem is to be solved, which means that pre-coded models are not called on.


Domain-specific language:

Such as SQL, is a type of computer language of limited scope, designed to address a particular problem within an application domain


Coding Princples

There are general rules and guidlines that programmers tend to follow. These makes syntax more universal as everyone pitches in to keep others updated on the most efficient way to code. Here are some examples of principles:


KISS: Keep it short and simple. Programs run best when code is concise and simple.

DRY: Dont Repeat Yourself. Using Functions/Loops is the efficient way of doing this and can cause less bugs and crashes.

Refactoring code: regularly examing and revising your code will make you a better programmer and make your program better.
It will take you less time to figure out errors.

Document your code: leaving comments that are glossed over by he machine allow you to keep track of your code and allows colleagues to examine your code more easily.

Key Terms

Algorithm: a set of instructions followed to solve a problem or process

Argument: a way to provide information to a function. Function then uses the information as it runs, like a variable.

Arrays: are containers that hold variables, used to group similar variables. It is a list with elements; the array is like a shelf and the things on it are the variables.

Arithmetic operators: is essential in applications. It does mathematic operations.
eg. If a character in a game earns XP it has to be added, if they take damage health has to be subtracted etc

Assignment operators: perform operations. For example the assignment operator "="" assigns it as equalled
Whereas the assignment operator "+"" will add them together etc.

IDE (Integrated Developemnt Environment): a software application that provides comprehensive facilities to computer programmers for software development.

Source code: the statements that are created by a programmer in the IDE.

Syntax: the way it has to be written to be understood by that language, i.e the grammar
